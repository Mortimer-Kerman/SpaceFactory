//Description des items
GameItems.d.foreuse : La foreuse est une machine servant à extraire du minerais automatiquement
GameItems.d.tapis : Les tapis roulants sont un merveilleux moyen de déplacer des items d'une machine vers une autre
GameItems.d.stockage : Le stockage permet, comme son nom l'indique, de stocker des items [les items sont stockés en dehors de l'inventaire, cliquez sur un stockage pour récupérer son contenu dans votre inventaire]
GameItems.d.jonction : La jonction permet de lier deux tapis roulant
GameItems.d.trieur : Permet de trier les différents items
GameItems.d.delete : Le menu de destruction permet de détruire un item
GameItems.d.charbon : Le charbon est un minerais qui peut servir de conbustible
GameItems.d.cuivre : Le cuivre est le plus ancien métal utilisé par l’Espèce humaine. Le point de fusion de ce minerais n'est pas trop élevé
GameItems.d.or : L'or est un excellent conducteur éléctrique, c'est un minerais apprécié pour son fort éclat de "petit soleil"
GameItems.d.m1 : M1, est un minerais, dont l'existence a été théorisé durtant les années 1950 via l'un des premier ordinateurs soviétique, le M-1
GameItems.d.error : Une erreur est survenue : Item introuvable/non documenté
